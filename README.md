# Website Saver App

This mobile app allows users to save websites device.

## Features
- Save website URLs with options to edit, delete, and visit
- Simple and intuitive user interface

## Installation

To run the app on your Android device:

1. Clone this repository to your local machine.
2. Install the required dependencies by running `npm install`.
3. Connect your Android device to your computer.
4. Run the app using Expo CLI by running `expo start` in the project directory.
5. Scan the QR code with the Expo Go app on your Android device.

## Usage

1. Open the app.
2. Enter the website URL.
3. Save the website.
4. Optionally, edit or delete saved websites.
5. Visit a saved website by tapping on it.

## Dependencies

- React Native
- Expo
- AsyncStorage
